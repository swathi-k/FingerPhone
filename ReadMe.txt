The FingerciseServer.java is in the Folder Server.

The Android application is in Android 3. Before Running Please, change the ip at the top of Scores.java, line 21.


Game 1:
Orientation Game
In this game, the orientation of the device has to be changed, in order to get points.


Game 2:
Image Game
In this game, the orientation of the picture has to be correct, in order to get points. The orientation can be changed, by rotating the device.


Game 3:
Light Bulb
In this game, the light bulb has to switched off/on using the button according to flashing text. If the correct action is done, users get points.
